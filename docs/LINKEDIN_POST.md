🚀 Excited to share my latest project: The Supermarket Price Index Pipeline!

📊 I built an end-to-end data engineering system that processes 630,000+ data points from major retailers (Metro, Al-Fatah, Chase Up) across 16 Pakistani cities.

Key engineering highlights:
✅ **Scalable Scraping**: Custom strategies for API traversal and Shopify fragment parsing.
✅ **Entity Resolution**: Advanced fuzzy matching using RapidFuzz to align product identities across stores.
✅ **Professional Analytics**: Implementation of Leader Dominance Index (LDI) and Spearman correlation for market behavior analysis.

This project goes beyond simple data collection—it's about building a robust, re-runnable pipeline that turns raw web fragments into executive-ready market insights.
